Project Summary

Archer was developed through a team of 4. We used Python, Flask, SQLAlchemy to create our website. The purpose of this website is to match you with other people who share either the same academic major, or a similar activity you both are interested in.

To achieve the match making, we simply had the user fill out a survey upon registering and the backend does a quiery through the database to find other accounts with matching survey data.

This project taught me the basics of:

  • Object Relational Mapping with SQLAlchemy
  • How to create a website through Flask-Python
  • HTML, CSS
  • Exposure to Agile and Scrum methodologies
