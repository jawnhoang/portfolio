Project Summary

JINA was a 4 person collaboration for my Object Oriented Class. We decided to test our skills and create a video game platform. Each person was responsible for a game and some extra features.

In order for all of us to achieve this project's goal, everyone utilized libraries, youtube tutorials, and the internet to their disposal in creating the games' foundation. It then became our liberty to tweak the games with our added features and whatever else we were able to squeeze in.

This project taught me the basics of:

  • JavaFX and how to use SceneBuilder
  • Basics of Object Oriented Designs, how to create proper classes
  • Exposure to Agile and Scrum methodologies
